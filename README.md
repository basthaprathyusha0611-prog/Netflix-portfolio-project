# Netflix-portfolio-project
This project focuses on performing Exploratory Data Analysis (EDA) on Netflix content data to uncover insights about movies and TV shows available on the platform.   It demonstrates practical data analysis skills using Python and visualization libraries, following an industry-relevant analytics workflow.
Project Overview

Netflix is one of the world’s leading streaming platforms with a vast content library.  
This project analyzes Netflix data to understand:
- Content distribution across countries
- Growth of content over the years
- Popular genres and ratings
- Differences between Movies and TV Shows

The goal is to convert raw data into meaningful insights that support content strategy and business decisions.

---
 Business Objective

To analyze Netflix’s content catalog and answer key questions such as:
- How has Netflix content grown over time?
- Which countries produce the most content?
- What are the most common genres?
- How does content differ by rating and type?

---

 Tools & Technologies Used

- Power BI for visualizing and Power Query for Transformation of data


---Project Workflow

Data Understanding
- Reviewed dataset structure
- Identified key columns such as:
  - Title
  - Type (Movie / TV Show)
  - Country
  - Release Year
  - Rating
  - Genre

---

Data Cleaning
- Handled missing values
- Removed duplicates
- Converted data types
- Standardized categorical columns

---

Exploratory Data Analysis (EDA)
Performed detailed analysis including:
- Movies vs TV Shows comparison
- Content growth by year
- Country-wise content distribution
- Genre and rating analysis

---

Data Visualization
Created insightful visualizations such as:
- Bar charts for content type distribution
- Line charts showing yearly growth
- Count plots for ratings and genres
- Country-wise content analysis

---

 Key Insights (Sample)

- Netflix has added content rapidly after 2015
- Movies dominate the platform compared to TV shows
- The United States contributes the highest number of titles
- Drama and International Movies are the most common genres
- TV-MA is the most frequent rating on Netflix
